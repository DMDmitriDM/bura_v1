### Карточная игра Бура (сборка Webpack).
* Правила игры могут отличаться от известных.

### Правила игры в данном проекте
* Число карт в колоде 36.
* 10 "Старше" всех карт, кроме Туза.
* Игра состоит из партий, каждая партия это новая сдача карт.
* За победу в партии противнику начисляется штраф "Борода" в кол-ве 2 очков.
* Если сумма "Бороды" у игрока достигает 6, то он считается проигравшим.
* При самой первой сдаче карт игрок который будет "ходить" первым оределяется жребием.
* В дальнейшем право первого хода определяется по победе в партии.
* Во течении партии право первого хода определяется за победу во взятке.
* Право первого хода может перейти к другому игроку если у него на руках окажется 4 карты одинаковой масти.
* Победа в партии определяется по очкам. Если очки одинаковые, то по кол-ву взяток в партии,
* если и они равны, тогда побеждает игрок у которого было право первого хода.

### Необходимые требования
node.js версия не ниже 22.12.0
npm версия не ниже 10.9.0
При необходимости установить serve
npm install --global serve

### Установка и запуск проекта
1. Склонируйте данный репозиторий к себе на диск.
2. Из директории bura_v1 выполните команду `npm i` для установки
и `npm run dev` для запуска проекта в режиме отладки.
Затем откройте браузер по адресу http://localhost:8080/.

3. Для создания продакшен проекта остановите сервер отладки командой CTRL+C
(если он был запущен).
4. Из директории bura_v1 выполните команду `npm run build`.
5. Из директории dist выполните команду `npx serve`.
Затем откройте браузер по адресу http://localhost:3000/.
